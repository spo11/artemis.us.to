In this guide I'll explain how to configure hyde once you've already installed it on your system. {% endmark -%} ### Installation If you haven't already installed python-pip, install it now. **On Arch Linux** 
sudo pacman -S python-pip
Then install hyde. **On Arch Linux** 
pip install hyde
